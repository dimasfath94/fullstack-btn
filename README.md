1. Soal Frontend (ReactJS/Next.js)
Soal:
Anda diminta untuk membuat komponen pencarian di aplikasi React. Komponen ini
harus dapat mencari produk berdasarkan nama dari sebuah array produk yang sudah
ada. Array produk terdiri dari objek dengan atribut name dan price.
Tugas:
1. Implementasikan komponen pencarian menggunakan React.
2. Hasil pencarian harus langsung ditampilkan pada layar.
3. Gunakan React hooks (useState dan useEffect).
4. Tampilan hasil pencarian harus menampilkan nama produk dan harga produk.
Contoh Array Produk:
const products = [
 { name: "Laptop", price: 1200 },
 { name: "Smartphone", price: 800 },
 { name: "Tablet", price: 600 },
];
2. Soal Backend (Node.js/Express.js)
Soal:
Buatlah API menggunakan Express.js yang menangani operasi CRUD untuk data
produk. Data produk memiliki atribut id, name, dan price. API tersebut harus memiliki
endpoint:
• GET /products: Mendapatkan semua produk.
• POST /products: Menambahkan produk baru.
• PUT /products/:id: Mengupdate produk berdasarkan id.
• DELETE /products/:id: Menghapus produk berdasarkan id.
3. Soal Database (MySQL)
Soal:
Tulis query SQL untuk menghitung total nilai pesanan (total_price) dari tabel orders
yang memiliki kolom order_id, customer_id, dan total_price, dimana Anda ingin
mengetahui total nilai pesanan yang dilakukan oleh setiap pelanggan (customer_id),
pada tahun 2025.
4. Soal Git
Soal:
Anda sedang bekerja di cabang feature-xyz. Tim lain telah menyelesaikan update besar
di cabang master. Sebelum Anda menggabungkan cabang feature-xyz dengan master,
tim meminta Anda untuk melakukan rebase terhadap cabang Anda agar bisa
digabungkan tanpa konflik. Jelaskan langkah-langkah yang harus Anda lakukan.
5. Soal SDLC (Waterfall dan Agile)
Soal:
Jelaskan perbedaan antara model Waterfall dan Agile dalam pengembangan perangkat
lunak, serta keuntungan dan kelemahan masing-masing model.
6. Soal PHP (CodeIgniter) – CRUD Produk
Soal:
Buat CRUD sederhana untuk tabel products (id, name, price) menggunakan
CodeIgniter.
7. Soal PHP (CodeIgniter) – Login Sederhana
Soal:
Buat fitur login sederhana menggunakan CodeIgniter dengan tabel users yang memiliki
field minimal username dan password.
Fitur harus mencakup:
• Proses autentikasi username dan password
• Penyimpanan session setelah login berhasil
• Fitur logout
